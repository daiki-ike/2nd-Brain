# 48歳サラリーマン、実は「適当」に生きてます。

見た目は真面目そうに見られるんです。
なんか「仕事できそう」とか言われるし。

でも中身は、マジで適当。
どれくらい適当かっていうと、明日提出の資料、まだ1行も書いてないくらい適当です。

ただ、小心者だから「怒られるのは嫌」なんですよ（ここ重要）。
人の目もめっちゃ気にするタイプだし。

だから、「怒られないギリギリのライン」を攻めるために、めちゃくちゃ効率化する。
自動化ツールとか使いまくって、一瞬で終わらせて、涼しい顔をするんです。

周りは「真面目だね」って言うけど、これ「全力でサボるための努力」なんですよね。
このパラドックス、伝わりますかね？

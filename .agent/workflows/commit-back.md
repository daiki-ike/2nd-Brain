---
description: コミットした時点まで元通りにする（Load Game）。
---

1.  **履歴の確認**:
    *   以下のコマンドを実行し、直近のコミット履歴を表示してください。
    *   `git log --oneline -n 10`

2.  **コミットの特定**:
    *   ユーザーが指定した「○○（キーワードや日時）」に該当するコミットを特定してください。
    *   もしユーザーの指定が曖昧な場合は、履歴を提示して「どの時点に戻しますか？」と聞いてください。

3.  **巻き戻し (Hard Reset)**:
    *   **警告**: 「この操作を行うと、指定した時点より後の変更はすべて削除され、元に戻せません。本当によろしいですか？」とユーザーに最終確認してください。
    *   承認が得られた場合のみ、以下のコマンドを実行して巻き戻してください。
    *   `git reset --hard <Commit_Hash>`
    *   ※ `<Commit_Hash>` は特定したコミットのID（例: `abc1234`）に置き換えてください。

4.  **強制同期 (Force Push)**:
    *   ローカルを巻き戻した後、リモート（GitHub）もその状態に合わせる必要があります。
    *   `git push -f origin main`
    *   実行後、「セーブポイントまで巻き戻しました！」と報告してください。

# 文体サンプル案 (Style Drafts)

頂いた自己分析「マジで適当だけど真面目」「仕組化が好き」「実は短気」をベースに、3つの媒体でサンプルを作成しました。
「これだ！」と思うものや、「ここは違う」という点を確認してください。

---

## 1. X (Twitter) ポスト案

### パターンA：効率化とサボりのパラドックス
ぶっちゃけ、仕事なんていかに「やらないか」だと思うんですよね。
サボりたいから、必死で仕組みを作る。
周りからは「仕事早くて真面目だね」って言われるけど、違うんよ。
早く終わらせて、ボーっとしたいだけなんよ。
このギャップ、誰にもバレてないことを祈る・・・ｗ

### パターンB：短気な内面の吐露（シャットダウン）
基本、仏のように我慢して仕事してるつもり。
でもさ、「それ、今言う？」みたいな理不尽なボール飛んできた瞬間、
心の中でシャッターが「ガシャン！」って閉まる音がする。
顔は笑顔のままだけど、もう心はシャットダウン済み。
このスイッチ入ったら、もうテコでも動きません。
大人だから我慢するけどさ・・・限界あるよね。

### パターンC：なんちゃって仕事できる風
「状況を読む」のが得意というか、人の顔色伺っちゃう性格なだけなんだけどね。
でもそれが「空気が読める」って評価になるなら、まあいいか。
面倒なタスクは全部ツールに任せて、人間関係の調整だけ適当にこなす。
これが僕の「なんちゃって仕事できる風」の正体です。
真面目にやるのは「仕組み作り」だけ。あとは流れでよろしく。

## 2. Note 記事の導入案

**タイトル：48歳サラリーマン、実は「適当」に生きてます。**

見た目は真面目そうに見られるんです。なんか「仕事できそう」とか言われるし。
でも中身は、マジで適当。
どれくらい適当かっていうと、明日提出の資料、まだ1行も書いてないくらい適当。

ただ、小心者だから「怒られるのは嫌」なんですよ（ここ重要）。
人の目もめっちゃ気にするタイプだし。

だから、「怒られないギリギリのライン」を攻めるために、めちゃくちゃ効率化する。
自動化ツールとか使いまくって、一瞬で終わらせて、涼しい顔をする。

周りは「真面目だね」って言うけど、これ「全力でサボるための努力」なんですよね。
このパラドックス、伝わるかなぁ・・・。
今日はそんな「なんちゃって仕事できる風」の裏側を、こっそり書いてみます。

## 3. メルマガ / エッセイ案

**件名：【告白】実は、キレやすいんです。**

こんばんは、大希です。

いつも「冷静ですね」とか言われるんですが、
実は僕、めちゃくちゃ沸点低いんですよ（笑）。

自分の思い通りにいかないと、すぐイラッとしちゃう。
頑固なところもあるから、一度「違う」って思うと譲れない。

でも、小心者だから表には出せないんですよね。
ぐっと我慢して、我慢して・・・
ある日突然、心が「プツン」と切れてシャットダウンする。

「あ、もう無理だわこれ」ってなったら、一気に冷める感じ。
皆さんにもありませんか？こういう瞬間。

そんな「面倒くさい自分」をどう扱うか？
僕なりの「自分の機嫌の取り方」について、今日は書いてみようかなと思います。
（まあ、そんな大した話じゃないんですけどね・・・）
